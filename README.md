# Privacy-and-Fraud-Detection-System
M7 Privacy & Fraud Detection System
The M7 Privacy & Fraud Detection System is a modular React-based platform offering integrated, role-based tools for both consumer privacy protection and enterprise-grade threat management.

Project Structure
This system is architected to separate concerns between consumer-facing privacy tools and business-level fraud analytics.

Consumer Hub
Trust Score: Real-time risk assessment via dynamic gauges.

Privacy Analyzer: URL-based audit of data tracking.

T&C Summarizer: Simplified breakdown of legal terms.

Browser Extension Simulator: Functional threat detection preview.

Business Console
Fraud Detection: Automated customer profiling with conditional risk-based highlighting.

Data Processing: Secure pipeline for CSV ingestion and deduplication.

Loyalty & Rewards: Customer management based on trust metrics.

Threat Analytics: Dashboard for visualizing security trends.

Tech Stack
Frontend: React, Tailwind CSS

Architecture: Modular React components

Setup
To run this project locally, ensure you have Node.js installed:

npm install

npm run dev
